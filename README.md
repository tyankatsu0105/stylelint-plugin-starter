# 🏁Stylelint starterkit plugin

Plugin starter kit for [Stylelint](https://stylelint.io/).

## Features

- [x] CLI
  - Hearing these info
    - [x] plugin name
    - [x] author
    - [x] yarn or npm
    - [x] scoped plugin
  - Update file with the above info
    - [x] package.json
    - [x] README.md
    - [x] yarn.lock or package-lock.json
- [x] test
- [x] rules
